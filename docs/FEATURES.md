| 相似度阈值过滤 | ✅已实现 | 设置页滑块调节（0–100%），低于阈值的结果不显示 |

| 批量多选 | ✅已实现 | Ctrl+点击 多选结果项 |

| 批量复制路径 | ✅已实现 | 批量复制选中结果的文件路径到剪贴板 |
| 图片对比 | ✅已实现 | 选择 2 张结果图片，分屏对比视图，同步缩放、拖拽平移、可拖拽分割线 |
| CSV 导出 | ✅已实现 | 导出含 img_id、来源类型、相似度、文件路径、UG 图号等字段 |
| 分页 | ✅已实现 | 超过 20 条结果时自动分页，支持上一页/下一页 |
| 相似度颜色标记 | ✅已实现 | >80% 绿色边框、50–80% 橙色边框、<50% 灰色边框 |
| 来源类型徽标 | ✅已实现 | 显示 Excel 内嵌 / UG 预览 / 文件图片 的来源类型标签 |
| OCR 文字显示 | ✅已实现 | 搜索结果中显示 OCR 提取的文字内容 |
| 右键上下文菜单 | ✅已实现 | 右键菜单：打开图片、打开 Excel、打开文件夹、复制路径、复制 UG 图号、切换收藏 |
| 悬停预览 | ✅已实现 | 鼠标悬停缩略图 400ms 后弹出 400×400 放大预览 |
| 懒加载缩略图 | ✅已实现 | IntersectionObserver 实现缩略图懒加载，损坏图片显示占位符 |
| 搜索历史 | ✅已实现 | 最多 20 条，含压缩缩略图(~10KB JPEG)，localStorage 持久化，点击重搜 |
| 模型加载进度 | ✅已实现 | AI 模型加载时显示进度条+状态文字，120s 超时后显示明确错误 |
| CLI 参数搜索 | ✅已实现 | `--search <path>` 命令行参数，应用启动时自动触发搜索 |
| 仪表板概览卡片 | ✅已实现 | 搜索页顶部显示资料库数量、索引图片数、最近扫描时间 |
