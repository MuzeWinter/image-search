## 1. 快速入门

### 1.1 安装

1. 下载最新 `ZOOBET-Image-Search_*.msi` 安装包
2. 双击安装，按向导完成
3. 安装包已内置 Python 运行时和 AI 模型，无需手动配置

### 1.2 首次启动

首次启动时会显示 **欢迎向导**，共 4 步：

1. **欢迎页** — 产品简介
2. **选择资料库** — 选择一个包含设计资料的文件夹
3. **首次扫描** — 自动扫描文件夹中的图片、Excel、UG 文件
4. **完成** — 开始使用

向导完成后不再显示。如需重新扫描或添加更多资料库，见 [3. 资料库](#3-资料库)。

### 1.3 界面概览

```
┌─────────────────────────────────────────────┐
│  顶部栏（窗口拖拽 / 最小化 / 最大化 / 关闭）    │
├──────────┬──────────────────────────────────┤
│          │                                  │
│  侧边栏   │         内容区域                  │
│  · 搜索   │                                  │
│  · 资料库 │                                  │
│  · 设置   │                                  │
│          │                                  │
├──────────┴──────────────────────────────────┤
│  底部状态栏（资料库数量 / 索引数 / 后台任务）    │
└─────────────────────────────────────────────┘
```

- **侧边栏**：点击切换搜索、资料库、设置三个页面
- **状态栏**：显示资料库状态、索引图片数、后台任务进度
- **快捷键**：`Ctrl+1` 搜索、`Ctrl+2` 资料库、`Ctrl+3` 设置

## 2. 搜索

### 2.1 以图搜图

ZOOBET 的核心功能：用一张图片搜索资料库中视觉相似的所有图片。

#### 三种输入方式

| 方式 | 操作 |
|------|------|
| **拖拽** | 从文件管理器拖一张图片到搜索区域 |
| **粘贴** | 复制图片后 `Ctrl+V` 粘贴到搜索区域 |
| **浏览** | 点击搜索区域，在文件选择器中选择图片 |

#### 搜索流程

1. 提供查询图片 → AI 模型提取视觉特征
2. 在资料库向量索引中搜索最相似的图片
3. 按相似度降序显示结果

首次搜索时需要加载 AI 模型（OpenCLIP ViT-B/32），约需 3-10 秒，进度条会显示加载状态。后续搜索即刻响应。

### 2.2 搜索结果

每条搜索结果包含：

- **缩略图** — 鼠标悬停 400ms 弹出 400×400 放大预览
- **文件名** — 原始文件名
- **来源类型标签** — Excel 内嵌 / UG 预览 / 文件图片
- **相似度** — 百分比 + 颜色标记（绿色 ≥80%、橙色 50-80%、灰色 <50%）
- **UG 图号** — 关联的 UG/CAD 编号
- **文件路径** — 完整路径，点击可复制
- **图片信息** — 尺寸、格式、文件大小
- **OCR 文字** — 图片中识别到的文字内容

### 2.3 过滤与排序

**来源过滤**：全部 / 仅 Excel 内嵌图 / 仅 UG 预览图 / 仅收藏

**资料库过滤**：在所有资料库中搜索，或限定到单个资料库

**相似度阈值**：设置页滑块调节 0-100%，低于阈值的结果不显示

**文本过滤**：在搜索结果中按文件名、UG 图号、OCR 文字、sheet 名二次过滤

**排序方式**：
- 相似度降序（默认）
- 文件名升序
- UG 图号升序

### 2.4 视图切换

**网格视图**：缩略图网格，适合快速浏览
- 缩略图尺寸三档可调：小(60px) / 中(100px) / 大(160px)
- 设置自动保存

**列表视图**：详细信息列表，适合查看完整元数据

### 2.5 图片对比

选中 2 张搜索结果，进入**分屏对比模式**：

- 左右分屏显示两张图片
- 支持同步缩放和拖拽平移
- 可拖拽中间分割线调整比例
- 方便与查询原图做细节对比

### 2.6 右键菜单

右键点击搜索结果弹出上下文菜单：

- 打开图片
- 打开所在 Excel 文件
- 在文件管理器中打开所在文件夹
- 复制文件路径
- 复制 UG 图号
- 切换收藏状态

### 2.7 收藏

点击结果上的星标图标切换收藏状态。收藏持久化保存，可通过来源过滤中的"仅收藏"筛选。

### 2.8 搜索历史

最近 20 条搜索记录，每条包含：
- 查询图片的压缩缩略图（约 10KB）
- 搜索时间和结果数

点击历史记录可重新执行该搜索。

## 3. 资料库

### 3.1 添加资料库

1. 切换到**资料库**页面（`Ctrl+2`）
2. 点击**添加资料库**按钮
3. 选择文件夹路径，或直接**拖拽文件夹**到资料库列表区域
4. 新资料库添加后状态为"就绪"

### 3.2 扫描资料库

点击资料库行的**扫描**按钮启动扫描。扫描分为 7 个阶段：

| 阶段 | 说明 |
|------|------|
| 1. 收集 | 递归遍历文件夹，收集所有文件 |
| 2. 哈希 | 计算文件 SHA256 哈希值 |
| 3. 比对 | 与数据库已有记录对比，识别变更 |
| 4. 移动检测 | 检测文件是否被移动到其他位置 |
| 5. 保存 | 将新文件和变更写入数据库 |
| 6. Excel 提取 | 从 .xlsx/.xls 中提取内嵌图片 |
| 7. UG 预览 | 通过 NXOpen 导出 .prt 预览图，自动索引 |

扫描进度实时显示：进度条、阶段名称、文件计数、已用时间、预估剩余时间、当前文件路径。

**取消扫描**：扫描过程中可随时点击取消按钮。

### 3.3 增量扫描

启用增量扫描后，系统只处理变更的文件（新增/修改/移除/移动），大幅减少重复扫描时间。

### 3.4 扫描结果

扫描完成后显示结果面板：

- 新增 / 修改 / 移除 / 移动 文件数量
- 扫描耗时
- 图片 / Excel / CAD / PDF 文件数量
- 自动关联匹配数

### 3.5 文件夹监控

在设置中启用**自动监控**后，系统通过操作系统文件事件实时监听资料库文件夹变更，5 秒防抖后自动触发增量扫描。

### 3.6 管理资料库

- **打开文件夹**：在系统资源管理器中打开资料库路径
- **删除**：删除资料库及其索引数据，5 秒内可点击**撤销**恢复
- **路径验证**：启动时自动检查路径是否存在，无效路径标记为"不可用"

## 4. 设置

### 4.1 外观

| 设置 | 选项 |
|------|------|
| **主题** | 浅色 / 深色 / 跟随系统 |
| **语言** | 中文 / 英文 |
| **强调色** | 蓝 / 绿 / 紫 / 橙 / 红 |

首次启动时自动检测系统语言和主题偏好。所有外观设置即时生效并持久化。

### 4.2 搜索设置

| 设置 | 说明 | 默认值 |
|------|------|--------|
| **相似度阈值** | 低于此值的结果不显示 | 0%（显示全部） |
| **返回数量 Top-K** | 搜索返回的最大结果数 | 20 |
| **OCR 开关** | 扫描时是否进行 OCR 文字识别 | 开启 |
| **UG 预览开关** | 扫描时是否提取 .prt 预览图 | 开启 |
| **UG 图号列名** | Excel 中存储 UG 图号的列名 | "图号" |

### 4.3 扫描设置

| 设置 | 说明 |
|------|------|
| **扫描扩展名** | 勾选预设扩展名（.xlsx/.xls/.prt）或添加自定义扩展名 |
| **自动监控** | 启用/禁用文件夹变更自动监控 |

### 4.4 数据库维护

| 操作 | 说明 |
|------|------|
| **备份数据库** | 备份 SQLite 数据库（含 WAL/SHM）到指定路径 |
| **恢复数据库** | 从备份文件恢复，恢复前自动备份当前数据库 |
| **数据库优化** | 执行 VACUUM + ANALYZE + REINDEX |
| **重建向量索引** | 清空所有向量嵌入，下次搜索时重建 FAISS 索引 |
| **清除缓存** | 删除所有临时文件 |
| **数据库大小** | 显示当前数据库文件大小（B/KB/MB） |

### 4.5 系统

- **系统诊断**：检查 Python 版本、pip 依赖、数据库完整性、磁盘空间、NXOpen 可用性、FAISS 索引状态
- **活动日志**：按级别过滤（全部/信息/警告/错误），查看时间戳、来源、消息
- **恢复默认设置**：一键重置所有设置为默认值，操作前二次确认
- **关于**：显示应用名称、版本号、技术栈徽标

## 5. 导出与分享

### 5.1 导出搜索结果

从搜索结果中导出：

| 导出方式 | 说明 |
|----------|------|
| **ZIP 打包** | 将选中结果的原图打包为 ZIP 文件 |
| **PDF 报告** | 生成含缩略图、相似度、路径的 PDF 报告 |
| **复制到剪贴板** | 复制选中图片到系统剪贴板 |
| **复制路径** | 复制选中结果的文件路径列表 |
| **在文件管理器中打开** | 在资源管理器中定位并选中文件 |
| **CSV 导出** | 导出元数据（img_id、来源类型、相似度、路径、UG 图号），BOM 兼容 Excel |

### 5.2 批量操作

- `Ctrl+点击` 多选结果
- 批量操作：导出、打开文件夹、复制路径、管理收藏

## 7. 常见问题

### 7.1 搜索没有结果

1. 确认资料库已完成扫描（状态栏显示索引数 > 0）
2. 降低相似度阈值（设置 → 相似度阈值）
3. 检查来源过滤是否限制了范围
4. 确认资料库中有图片文件

### 7.2 AI 模型加载失败

1. 检查 Python 依赖：`pip install -r requirements.txt`
2. 检查磁盘空间（模型文件约 600MB）
3. 首次加载需要下载模型，确保网络连接
4. 查看设置 → 系统诊断 → Python 依赖检查

### 7.3 扫描很慢

1. 首次全量扫描较慢，后续增量扫描会快很多
2. 可以关闭不需要的功能（OCR、UG 预览）加快速度
3. 减少扫描扩展名数量

### 7.4 UG .prt 文件没有预览图

UG 预览图提取需要安装 Siemens NX 软件。如未安装 NX，可在设置中关闭 UG 预览开关。

### 7.5 软件启动后资料库显示"不可用"

资料库路径已被移动或删除。点击资料库行查看原路径，重新添加正确的文件夹路径。

### 7.6 如何迁移数据到新电脑

1. 在旧电脑设置 → 数据库维护 → 备份数据库
2. 将备份文件复制到新电脑
3. 在新电脑设置 → 数据库维护 → 恢复数据库
4. 重新添加资料库路径（路径变化时需重新扫描）

### 7.7 数据安全

所有数据存储在本地，不上传任何内容到云端。AI 模型在本地运行，查询图片不离开本机。
