# Claude Code 任务单：v2-21 — 搜索历史记录

遵守全部规则文件：AGENTS.md > CLAUDE.md > docs/AI-CODING-RULES.md

## 目标
记录最近搜索的图片（缩略图+时间），方便用户快速回溯。

## 必须实现

### 1. 存储
- localStorage 存储最近 20 条搜索记录
- 每条记录：缩略图 base64（限制 10KB）、时间戳、结果数量

### 2. UI (Search.tsx)
- 搜索框下方显示历史缩略图横排列表
- 点击历史缩略图重新搜索
- 右键可删除单条
- "清除全部"按钮（搜索结果为 0 或首次使用时隐藏）

### 3. 不破坏原有功能
- 不影响现有拖拽搜索
- 不影响现有粘贴搜索

## 验收标准
- `npm run build` 零错误
- 搜索后历史列表更新
- 关闭重开后历史保留
