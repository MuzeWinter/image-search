## 必须实现

### 1. CAD 文件页面
- 列表显示：CAD 编号、文件名、路径、类型、大小、关联图片数
- 筛选：按扩展名（prt/step/igs/dwg/...）
- 操作：打开文件 / 打开文件夹 / 复制路径
- 关联图片预览

### 2. Excel 记录页面
- 列表显示：EX 编号、文件名、工作表、行号、关联图片
- 点击行展开详情
- 操作：打开 Excel / 打开文件夹 / 复制路径

### 3. PDF 文件页面
- 列表显示：DOC 编号、文件名、页数、大小
- 预览图显示
- 操作：打开 PDF / 打开文件夹

### 4. 匹配管理页面
- 三种视图：自动匹配 / 疑似匹配 / 未匹配
- 确认 / 拒绝 / 手动绑定 操作
- 匹配方法标注：excel-reference / filename-match / same-folder / ai-similar / manual-bind
- 人工确认过的不被自动覆盖

### 5. 标签分类页面
- 标签云 / 列表切换
- 按标签筛选图片
- 新建/编辑/删除标签
- 标签颜色

### 6. 收藏夹页面
- 收藏的图片列表
- 取消收藏
- 筛选/排序
